Discuss the scope of var, let, and const

  • var এর স্কোপ হলো ফাংশন স্কোপ এবং ফাংশনের ভিতর থেকেই অ্যাক্সেস করা যায়। তবে, যদি var দিয়ে একটি ভ্যারিয়েবল ফাংশনের বাইরে ডিক্লেয়ার করা হয়, তাহলে সেটি গ্লোবাল স্কোপের হিসাবে অ্যাক্সেস করা যাবে।
  • let এবং const এর স্কোপ হলো ব্লক স্কোপ। এরা শুধুমাত্র ডিক্লেয়ার করা ব্লকের ভিতরেই অ্যাক্সেস করা যায়।
  • const দিয়ে ডিক্লেয়ার করা ভ্যারিয়েবল পুনরায় অ্যাসাইন করা যাবেনা, কিন্তু let দিয়ে ডিক্লেয়ার করা ভ্যারিয়েবলের মান পরিবর্তন করা যাবে। var দিয়ে ডিক্লেয়ার করা ভ্যারিয়েবলগুলির মান পরিবর্তনিতেও ব্যবহার করা যায়, তবে hoisting-এর জন্য var-এর ব্যবহার থেকে বিরত থাকা উচিত।

Tell us the use cases of null and undefined

  • undefined হলো একটি ডেটা টাইপ যা একটি ভ্যারিয়েবলে ভ্যালু না থাকলে দেখা যায়। যেমন, একটি ভ্যারিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু সেটি কোনো ভ্যালু অ্যাসাইন করা হয়নি।
  • null হলো একটি প্রিমিটিভ ভ্যালু যা একটি ভ্যারিয়েবলের মান হিসেবে অ্যাসাইন করা হয়। null দিয়ে একটি ভ্যারিয়েবলের "নাল" (নিখোজ) ভ্যালু সেট করা যায়।।
  • undefined এর ব্যবহার: আমরা undefined ব্যবহার করে একটি ভ্যারিয়েবলের মান চেক করতে পারি।
  • null এর ব্যবহার: null ব্যবহার করে আমরা একটি ভ্যারিয়েবলের "নিখোজ" (empty) ভ্যালু সেট করতে পারি।

What do you mean by REST API?

  • REST API হলো Representational State Transfer Application Programming Interface (API)। এটি একটি ওপেন স্ট্যান্ডার্ড API যা ওয়েব অ্যাপ্লিকেশন ডেভেলপমেন্ট এর জন্য ব্যবহৃত হয়।
  • REST API এর উদ্দেশ্য হলো ওয়েব সার্ভার থেকে ডেটা গ্রহণ করা। REST API এর মাধ্যমে, একটি ওয়েব অ্যাপ্লিকেশন অন্য একটি ওয়েব অ্যাপ্লিকেশনের সাথে ডেটা শেয়ার করতে পারে।
  • REST API এর URI (Uniform Resource Identifier) এর মাধ্যমে URL (Uniform Resource Locator) বা URN (Uniform Resource Name) দিয়ে HTTP Request পাঠানো হয়। URI এর মাধ্যমে, REST API এর Endpoint (API URL) স্পষ্টীকরন করা হয়।
